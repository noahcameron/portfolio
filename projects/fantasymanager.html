Automated Fantasy Hockey Manager

  • Tools/Methods: Python, PostgreSQL, Yahoo Fantasy Sports API, Google Apps Script, HTML, JavaScript)
  • Full Project Details: Project Link

Created an Automated Fantasy Hockey Manager to automate users' Yahoo Fantasy Sports Teams, comprising of 3 separate parts:

1. RDBMS

Created a PostgreSQL database to maintain all relevant data - player info, player stats, fantasy ownership, NHL teams, NHL schedules, fantasy week schedules, matchup scores, lineups. Created and scheduled python scripts to pull data from the Yahoo Fantasy Sports API nightly, to keep PostgreSQL database up to date.

2. Automated Decision Making for Fantasy Managers

Automated Decision Making for Fantasy ManagersCreated an object oriented Python program to manage players, teams, and weekly lineups. The program determines the best pickups to make each week, the best players to start in your lineup, and uses the Yahoo Fantasy Sports API to automatically make changes directly on the app.

3. Hubpage

Created a hub page using HTML and Google Apps Scripts, where users can sign into their team, and make changes to which players the program can consider dropping that week. They can also view most recent recommendations and projections: suggested lineup, planned pickups, projected matchup outcomes, and top players to target that week.
